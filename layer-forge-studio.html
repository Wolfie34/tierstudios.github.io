Technical Details

Core System

  • Layer-based image and texture editing inside Unity
  • Real-time editing with instant feedback
  • Non-destructive workflow with re-editable structure
  • Full Undo / Redo system

Editing Workflow

  • Right Click → Edit → Apply workflow
  • Direct editing without external tools
  • Seamless pipeline integration

Save & Data System

  • Auto-save to prevent data loss
  • Full project save via .lfs files
  • Smart re-editing via .lfsmeta files
  • Reopen and continue editing modified images

Advanced Layer System

  • Multi-layer editing & Layer grouping
  • Clipping masks & Layer masks
  • Adjustment layers (non-destructive)
  • Opacity, blending controls & ordering

Effects & Tools

  • Blur, Glow, Shadow, Color/Gradient Overlay
  • Move (Auto-Select), Brush, Clone Stamp, Smudge
  • Shape, Pen, Crop, Marquee/Lasso, Fill, Text
  • Transform (move, scale, rotate, perspective warp)

Compatibility & Performance

  • Optimized GPU-assisted real-time editing
  • Works with standard texture formats (PNG, JPEG)
  • Compatible with Built-in, URP and HDRP
  • Unity Editor tool (no runtime dependency)

Description

Edit, design and enhance your images directly inside Unity — no external tools, no broken workflow.

Layer Forge Studio is a powerful, layer-based image and texture editor built for Unity. It allows you to modify, create and refine visuals in real-time without ever leaving the engine.

Whether you're polishing UI, creating textures, or making quick adjustments to assets, Layer Forge Studio gives you a fast, intuitive and complete editing workflow.
