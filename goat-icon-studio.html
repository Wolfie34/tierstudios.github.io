Description

Create game-ready 2D icons from your 3D prefabs directly inside the editor.

GOAT Icon Studio is designed to speed up your UI workflow by turning models into clean, production-ready icons in seconds — no external tools, no manual screenshots.

Generate single icons or entire icon sets with a fast and intuitive workflow. Everything is handled inside the editor, so you can focus on creating, not exporting.

Key Features

  • One-Click Generation: Support for Single & Batch icon generation.
  • Animation-Based Capture: Choose the perfect pose directly from animations.
  • Multi-Object Composition: Combine multiple assets into one icon easily.
  • Real-Time Preview: Rotate, zoom, and frame with mouse control instantly.
  • Flexible Backgrounds: Use transparent, color, gradient, or image backgrounds.
  • Shader-Based Effects: Built-in Outline & Glow effects to make icons pop.
  • Adjustable Lighting: Fine-tune lighting color, intensity, and angle.
  • High-Quality Export: Export as PNG (ready for UI use).
  • Auto Sprite Setup: Drag & drop ready immediately after export.

Use Cases

Perfect for:

  • Inventory systems
  • RPG & strategy UI
  • Card and item icons
  • Shop and marketplace assets
  • Character portraits
