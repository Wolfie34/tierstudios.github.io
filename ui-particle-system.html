Description

Create UI particle effects directly on Canvas — no sorting issues, no hacks, no complexity.

UI Particle System is a high-performance solution built specifically for UGUI, allowing you to create clean, responsive, and professional UI effects with a familiar workflow.

Unlike traditional particle systems, this solution renders natively inside the Canvas hierarchy. Your particles behave like real UI elements, eliminating sorting layer conflicts and camera-based workarounds.

The workflow feels similar to Unity's Particle System, so you can start creating immediately without a learning curve.

Key Features

  • Native Canvas Rendering: No Sorting Issues — particles live inside the UI hierarchy.
  • Familiar Workflow: Shuriken-like interface means zero learning curve.
  • Mobile Optimized: Lightweight Zero-GC Core for smooth performance even on low-end devices.
  • Built-in Sound Effects: Trigger sounds on Birth, Death, Start, and Stop events.
  • Full Masking Support: Compatible with RectMask2D & UI Mask.
  • Advanced Modules: Trails, Noise, Attractors, Sub Emitters and more.
  • Texture Sheet Animation: Sprite-based animated particles.
  • All Canvas Modes: Works with Overlay, Camera, and World Space.
  • 100+ Ready Prefabs: Start creating effects immediately with a large prefab library.
